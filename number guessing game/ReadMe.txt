Number Guessing Game
Welcome to the Number Guessing Game project by TechnoHacksEduTech! This README file provides all the essential information for understanding and running the game.

Project Overview
The Number Guessing Game is a simple and fun project where the player tries to guess a randomly generated number within a specified range. The game is designed to enhance problem-solving skills and intuition while providing an enjoyable gaming experience.

How to Play:
The computer will generate a random number between a minimum and maximum range.
The player's task is to guess the correct number within that range.
After each guess, the game will provide feedback, indicating whether the guessed number is too high or too low.
The player will continue guessing until the correct number is guessed.
The game will display the number of attempts it took to guess the correct number.
The player can choose to play again if desired.
