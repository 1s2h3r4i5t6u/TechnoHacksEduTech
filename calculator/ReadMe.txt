TechnoHacksEduTech Calculator Project
Welcome to the TechnoHacksEduTech Calculator project! This README file will provide you with all the necessary information to understand and use the calculator.

Project Overview
The TechnoHacksEduTech Calculator is a simple yet powerful calculator application that performs basic arithmetic operations. It is designed to help users with their mathematical calculations and is a useful tool for students, professionals, and anyone in need of quick computations.

Features
Addition: Perform addition of two or more numbers.
Subtraction: Perform subtraction of two numbers.
Multiplication: Perform multiplication of two or more numbers.
Division: Perform division of two numbers.
Clear Function: Clear the calculator display.
Error Handling: Display meaningful error messages for invalid inputs.
